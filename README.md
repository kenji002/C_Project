# C_Project - シンプル電卓プログラム

C言語で作成された、標準入出力を用いたシンプルな電卓プログラムです。

## 機能

- **四則演算**: 加算(`+`)、減算(`-`)、乗算(`*`)、除算(`/`)に対応しています。
- **エラーハンドリング**: 0での除算を試みた場合に警告を表示します。

## 使い方

### 1. コンパイル
ターミナルで以下のコマンドを実行してコンパイルします。
```bash
gcc calculator.c -o calculator.exe
```

### 2. 実行
コンパイル後、以下のコマンドで実行します。
```bash
./calculator.exe
```

### 3. 操作方法
1. 1つ目の数字を入力し、Enterを押します。
2. 演算子（`+`, `-`, `*`, `/`）を入力し、Enterを押します。
3. 2つ目の数字を入力し、Enterを押します。
